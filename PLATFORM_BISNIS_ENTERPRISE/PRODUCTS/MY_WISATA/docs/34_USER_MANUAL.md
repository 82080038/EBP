# MODUL 34 — USER MANUAL (WISATAWAN)

> **Versi:** 1.0 · **Tanggal:** 2026-06-30

---

## 1. RINGKASAN

Panduan penggunaan aplikasi Tour Guide untuk wisatawan.

---

## 2. PENDAFTARAN & LOGIN

### 2.1 Membuat Akun

1. Buka aplikasi di https://tourguide.com
2. Klik tombol "Daftar" di pojok kanan atas
3. Isi formulir pendaftaran:
   - Nama lengkap
   - Email
   - Password (minimal 8 karakter)
   - Konfirmasi password
4. Klik "Daftar"
5. Cek email untuk verifikasi
6. Klik link verifikasi di email

### 2.2 Login

1. Klik tombol "Masuk"
2. Masukkan email dan password
3. Klik "Masuk"
4. Anda akan diarahkan ke dashboard wisatawan

### 2.3 Lupa Password

1. Klik "Lupa Password" di halaman login
2. Masukkan email terdaftar
3. Cek email untuk link reset password
4. Buat password baru
5. Login dengan password baru

---

## 3. DASHBOARD WISATAWAN

### 3.1 Menu Utama

- **Beranda**: Ringkasan aktivitas dan rekomendasi
- **Cari Guide**: Pencarian tour guide
- **Booking Saya**: Daftar booking tour guide
- **Tiket Saya**: E-ticket destinasi wisata
- **Hotel**: Pencarian dan booking akomodasi
- **Restoran**: Pemesanan makanan
- **Event**: Daftar dan pendaftaran event budaya
- **Audio Guide**: Panduan audio destinasi
- **AI Assistant**: Chatbot bantuan
- **Notifikasi**: Pesan dan notifikasi
- **Profil**: Pengaturan profil

### 3.2 Ringkasan Dashboard

Dashboard menampilkan:
- Booking aktif
- Tiket yang akan digunakan
- Event yang akan dihadiri
- Rekomendasi destinasi dan guide

---

## 4. MENCARI TOUR GUIDE

### 4.1 Pencarian Basic

1. Klik menu "Cari Guide"
2. Masukkan lokasi atau nama guide di kolom pencarian
3. Klik "Cari"
4. Hasil pencarian akan ditampilkan

### 4.2 Filter Lanjutan

Filter yang tersedia:
- **Lokasi**: Pilih kota atau area
- **Bahasa**: Bahasa yang dikuasai guide
- **Spesialisasi**: Wisata alam, budaya, kuliner, dll
- **Harga**: Range harga per jam
- **Rating**: Minimal rating guide
- **Ketersediaan**: Tanggal yang diinginkan

### 4.3 Melihat Profil Guide

1. Klik pada guide dari hasil pencarian
2. Profil guide akan ditampilkan:
   - Foto dan nama
   - Rating dan ulasan
   - Bahasa yang dikuasai
   - Spesialisasi
   - Harga per jam
   - Jadwal ketersediaan
   - Ulasan dari wisatawan lain

---

## 5. BOOKING TOUR GUIDE

### 5.1 Membuat Booking

1. Pilih guide yang diinginkan
2. Klik "Booking Guide"
3. Isi formulir booking:
   - Tanggal tour
   - Jam mulai
   - Durasi (jam)
   - Jumlah peserta
   - Catatan khusus (opsional)
4. Klik "Lanjut ke Pembayaran"
5. Pilih metode pembayaran
6. Selesaikan pembayaran
7. Booking code akan ditampilkan

### 5.2 Status Booking

- **Pending**: Menunggu konfirmasi guide
- **Confirmed**: Guide telah menerima booking
- **Completed**: Tour telah selesai
- **Cancelled**: Booking dibatalkan
- **Rejected**: Guide menolak booking

### 5.3 Membatalkan Booking

1. Buka menu "Booking Saya"
2. Cari booking yang ingin dibatalkan
3. Klik "Batalkan"
4. Pilih alasan pembatalan
5. Konfirmasi pembatalan
6. Refund akan diproses sesuai kebijakan

---

## 6. MEMBELI TIKET DESTINASI

### 6.1 Mencari Destinasi

1. Klik menu "Tiket Wisata"
2. Gunakan filter untuk mencari:
   - Kategori wisata
   - Lokasi
   - Harga tiket
3. Klik destinasi untuk melihat detail

### 6.2 Membeli Tiket

1. Pilih destinasi
2. Pilih jenis tiket:
   - Dewasa
   - Anak-anak
   - Senior
3. Masukkan jumlah tiket
4. Pilih tanggal kunjungan
5. Klik "Beli Tiket"
6. Pilih metode pembayaran
7. Selesaikan pembayaran
8. QR code tiket akan ditampilkan

### 6.3 Menggunakan Tiket

1. Buka menu "Tiket Saya"
2. Pilih tiket yang akan digunakan
3. Tunjukkan QR code di lokasi destinasi
4. Petugas akan memindai QR code
5. Tiket akan ditandai sebagai "used"

---

## 7. BOOKING HOTEL/HOMESTAY

### 7.1 Mencari Akomodasi

1. Klik menu "Hotel"
2. Masukkan lokasi dan tanggal
3. Filter berdasarkan:
   - Tipe akomodasi (hotel, homestay, villa)
   - Harga per malam
   - Fasilitas
   - Rating
4. Lihat detail akomodasi

### 7.2 Booking Kamar

1. Pilih akomodasi
2. Pilih tipe kamar
3. Masukkan tanggal check-in dan check-out
4. Pilih jumlah kamar dan tamu
5. Klik "Booking"
6. Selesaikan pembayaran
7. Konfirmasi booking akan dikirim via email

---

## 8. PEMESANAN RESTORAN

### 8.1 Mencari Restoran

1. Klik menu "Restoran"
2. Cari berdasarkan lokasi atau jenis makanan
3. Lihat menu dan harga
4. Baca ulasan pelanggan

### 8.2 Memesan Makanan

1. Pilih restoran
2. Tambahkan menu ke keranjang
3. Lihat keranjang dan total harga
4. Pilih tipe pesanan:
   - Dine-in
   - Takeaway
   - Delivery
5. Klik "Checkout"
6. Selesaikan pembayaran
7. Tunggu konfirmasi restoran

---

## 9. PENDAFTARAN EVENT

### 9.1 Melihat Event

1. Klik menu "Event"
2. Lihat kalender event
3. Filter berdasarkan:
   - Kategori event
   - Tanggal
   - Lokasi
4. Klik event untuk detail

### 9.2 Mendaftar Event

1. Pilih event yang diinginkan
2. Cek ketersediaan kuota
3. Klik "Daftar"
4. Isi data pendaftaran
5. Bayar jika event berbayar
6. Konfirmasi pendaftaran akan dikirim
7. Notifikasi pengingat akan dikirim H-1

---

## 10. AUDIO GUIDE

### 10.1 Menggunakan Audio Guide

1. Kunjungi destinasi wisata
2. Buka menu "Audio Guide"
3. Pilih destinasi yang sedang dikunjungi
4. Pilih bahasa yang diinginkan
5. Putar audio guide
6. Baca transkrip jika tersedia

### 10.2 Fitur Audio Player

- Play/Pause
- Seek forward/backward
- Volume control
- Pilihan bahasa
- Transkrip teks

## 11. AI ASSISTANT

### 11.1 Menggunakan AI Assistant

1. Klik menu "AI Assistant"
2. Ketik pertanyaan di kolom chat
3. AI akan memberikan jawaban
4. Contoh pertanyaan:
   - "Destinasi wisata apa yang direkomendasikan di Yogyakarta?"
   - "Bagaimana cara ke Borobudur dari Malioboro?"
   - "Tour guide bahasa Inggris yang tersedia besok?"

### 11.2 Fitur AI Assistant

- Rekomendasi destinasi
- Informasi rute
- Tips wisata
- Bantuan booking
- FAQ umum

### 13.1 Mengedit Profil

1. Klik menu "Profil"

3. Update informasi:
   - Nama
   - Email
   - Nomor telepon
   - Foto profil

### 13.2 Mengubah Password

1. Buka menu "Profil"
2. Klik "Ubah Password"
3. Masukkan password lama
4. Masukkan password baru
5. Konfirmasi password baru
6. Klik "Simpan"

### 13.3 Pengaturan Notifikasi

1. Buka menu "Profil"
2. Klik "Pengaturan Notifikasi"
3. Pilih notifikasi yang ingin diterima:
   - Email
   - SMS
   - In-app

## 14. ULASAN & RATING

### 14.1 Memberikan Ulasan

1. Buka menu "Booking Saya" atau "Tiket Saya"
2. Pilih booking/tiket yang selesai
3. Klik "Beri Ulasan"
4. Berikan rating (1-5 bintang)
5. Tulis ulasan
6. Upload foto (opsional)
7. Klik "Kirim"

### 14.2 Melihat Ulasan Saya

1. Buka menu "Profil"
2. Klik "Ulasan Saya"
3. Lihat semua ulasan yang telah diberikan

## 16. TIPS PENGGUNAAN

### 16.1 Tips Booking

- Booking guide minimal 3 hari sebelum tanggal
- Cek ketersediaan guide sebelum booking
- Baca ulasan guide sebelum memilih
- Konfirmasi detail tour dengan guide

### 16.2 Tips Pembayaran

- Gunakan metode pembayaran yang terverifikasi
- Simpan bukti pembayaran
- Cek email konfirmasi setelah pembayaran

### 16.3 Tips Kunjungan

- Bawa QR code tiket saat kunjungan
- Cek jadwal dan lokasi sebelum berangkat
- Siapkan dokumen yang diperlukan (KTP, dll)
- Patuhi aturan destinasi wisata

## 17. KEAMANAN

### 17.1 Tips Keamanan

- Jangan bagikan password ke siapapun
- Logout setelah menggunakan aplikasi di perangkat publik
- Gunakan password yang kuat
- Update password secara berkala
- Laporkan aktivitas mencurigakan

### 17.2 Melaporkan Masalah

Jika mengalami masalah:
1. Buka menu "Bantuan"
2. Klik "Laporkan Masalah"
3. Jelaskan masalah secara detail
4. Screenshot jika perlu
5. Kirim laporan
